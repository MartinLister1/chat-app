# Chat App

A real time chat app built with Python, Flask and WebSockets.

Users enter a name to join the chat and can send messages instantly 
to everyone else in the room. A live user count updates as people 
join and leave. No login required and no message history is saved.

## Built with
- Python
- Flask
- Flask-SocketIO
- HTML, CSS, JavaScript

## Live demo
https://chat-app-h7hh.onrender.com

Note: hosted on Render's free tier so may take ~50 seconds to wake 
up if it hasn't been used recently.

## Run locally
1. Clone the repo
2. Install dependencies: pip install flask flask-socketio
3. Run: python app.py
4. Open http://localhost:5000
